# Uri1040

Conceito

Estrutura condicional:

É uma estrutura de controle
que permite definir que um
certo bloco de comandos
somente será executado
dependendo de uma condição.

COMPOSTA

if ( < condição> ) {
     <comando 1>
	 <comando 1>
}	 
else {
     <comando 3>
	 <comando 4>
}

Encadeamento de estruturas condicionais

if ( condição 1 ) {
  comando 1
  comando 2
}
else if ( condição 2 ) {
  comando 3
  comando 4
}
else if ( condição 3 ) {
  comando 5
  comando 6
}
else {
  comando 7
  comando 8
}
